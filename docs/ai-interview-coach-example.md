# AI Interview Coach using Endee

This is an example project demonstrating how Endee can be used in a real-world AI application.

## Project Overview

AI Interview Coach is a full-stack application that helps users prepare for coding interviews using machine learning.

## How Endee Can Be Used

Endee can enhance this project by:

- Storing vector embeddings of coding problems
- Performing similarity search for recommendations
- Improving personalization of problem suggestions

## Example Flow

1. Convert problem description to vector
2. Store vectors in Endee
3. Query similar problems based on user performance
